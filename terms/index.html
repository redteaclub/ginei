# 銀河帝國：

魯道夫．馮．高登巴姆於宇宙曆３１０年，以銀河聯邦首相兼國家元首的身份，篡奪了銀河聯邦政府，登基為帝，是為「銀河帝國．高登巴姆王朝」。後來新皇帝萊因哈特．馮．羅嚴克拉姆篡奪了「高登巴姆王朝」，以新王朝──「羅嚴克拉姆王朝」──的形式延續銀河帝國的歷史。

# 高登巴姆王朝：

魯道夫．馮．高登巴姆篡奪了銀河聯邦政府，是為「銀河帝國」。他及其子孫之王朝被稱為「高登巴姆王朝」，也有稱為「黃金樹王朝」。王朝共有三十八個皇帝，由宇宙曆３１０年起到宇宙曆７９９年終，國祚四百九十年。

# 羅嚴克拉姆王朝：

新皇帝萊因哈特．馮．羅嚴克拉姆，於帝國曆４９０年，宇宙曆７９９年，篡奪了「高登巴姆王朝」，創立的新王朝是為「羅嚴克拉姆王朝」。同年改行新帝國曆，是為新帝國曆元年。

# 自由行星同盟：

帝國曆１６４年，自由行星同盟國父──亞雷．海尼森帶領共和主義者們長征一萬光年，在帝國曆２１８年到達了壯年期星群的地帶，並在此建立了根據地，「自由行星同盟」成立，以共和民主政治的形式，對抗銀河帝國的獨裁帝政，時值宇宙曆５２７年。

宇宙曆７９９年，尚未篡位登基的萊因哈特．馮．羅嚴克拉姆公爵，以帝國宰相兼帝國三長官之姿，遠征自由行星同盟，同盟國在城下之盟的情況下與銀河帝國簽訂了「巴拉特和約」，自由行星同盟名存實亡。宇宙曆８００年，新皇帝萊因哈特以銀河帝國高等事務官雷內肯普被俘自殺一事為由，向自由行星同盟再次用兵，同盟軍在馬爾．亞迪特星域被帝國軍殲滅。其後皇帝萊因哈特在同盟國立美術館公布了「冬薔薇園敕令」，宣佈自由行星同盟正式滅亡，國祚二百七十三年。

# 費沙自治領：

一個夾處於帝國及同盟之間的都市國家，屬於費沙恆星的星系，直隸於銀河帝國，須對帝國納貢，但內政上卻擁有完全的自由權，而其中更包括了對自由行星同盟的外交和通商，人口約２０億。自治領主隸屬於皇帝，統治自治領土，並負責監督與同盟之間的交易，有時也身兼外交官的職務。由於其獨攬交易大權，財富不斷累積擴充，因此統轄的領域雖小，但其實力卻輕忽不得。

其實費沙自治領的成立，是由地球教所促成的。帝國曆３７３年，出身於地球的費沙商人雷歐波特．拉普在地球資本的協助下，向銀河帝國要求自治，費沙在地球資本和地球教宗教勢力的協助下而成功自治。故費沙與地球教關係極為密切，費沙可說是地球教在經濟等事上在地面的勢力。

帝國曆４８９年、宇宙曆７９８年，萊因哈特．馮．羅嚴克拉姆遠征自由行星同盟，是為「諸神之黃昏」作戰，行星費沙被帝國軍佔領，自治領主魯賓斯基失蹤。費沙自治領在傀儡領主博爾德克的管理下，臣服於銀河帝國，名存實亡。後來博爾德克在魯賓斯基的陰謀下，被國家安全保障局的朗古陷害而枉死獄中，皇帝萊因哈特便配合新政策而遷都費沙，費沙自治領正式滅亡，國祚一百一十七年。

# 奧丁：

瓦爾哈拉星域的第三行星──奧丁，是銀河帝國的首都，自魯道夫大帝建立銀河帝國後遷都於此，四百九十年間一直是「銀河帝國．高登巴姆王朝」的首都。高登巴姆王朝所有重要樞紐，如新無憂宮、宰相府、軍務省等均設置於此。新皇帝萊因哈特篡位後才遷都往費沙。其名字出於北歐神話（註：銀河帝國的社會文化是以德國日耳曼的文明為主，而人民和貴族的姓名也有著德國風格）的主神──奧丁。

# 海尼森：

自由行星同盟的首都，是巴拉特星域的第四行星，其名是紀念國父亞雷．海尼森。同盟所有重要樞紐，如最高評議會、統合作戰本部等均設置於此。衛星軌道上共十二個的迎擊衛星，名為「處女神之首飾」。宇宙曆８００年，自由行星同盟正式滅亡，海尼森作為同盟首都的任務結束，成為了新銀河帝國的轄下星球。到了宇宙曆８０１年，伊謝爾倫共和政府與皇帝萊因哈特達成和議，海尼森與其所屬的巴拉特星系成為了共和政治的最後根據地。

# 銀河聯邦：

西元２８０１年，政治統一中樞由太陽系第三行星地球，遷移至畢宿五（金牛座α）第二行星德奧里亞。人類發表銀河聯邦創立宣言，同年改為宇宙曆元年，開始向銀河系深處拓展。宇宙曆３１０年，魯道夫．馮．高登巴姆以銀河聯邦首相兼國家元首的身份，篡奪了銀河聯邦政府，聯邦政府於焉滅亡。

# 魯道夫．馮．高登巴姆：

宇宙曆２６８年出生於軍人家庭，十九歲時以首席的名次於宇宙軍預官學校中畢業。身高一九五公分，體重九十九公斤，二十歲時任少尉軍官階。配屬於航路警備部隊時，銳意整肅部隊內部綱紀，即使是上級也毫不留情。上級惹他不起而讓其晉升中尉，調派至宇宙海盜猖獗的地帶。乘興而來的魯道夫展現其強硬的作風，投降和候審的人也悉數燒死，雖引來批評，但群眾頌揚聲浪卻更高。

宇宙曆２９６年，二十八歲的魯道夫少將卸下軍籍轉入政界，在議會取得一席之地後，成為了「國家革新同盟」的領袖。其聲望號召了不少的年輕政治家，經過幾次選舉後終於成功奠定了政治的基礎。他根據國民投票成為首相後，利用憲法中沒有明文規定禁止兼任的漏洞，透過議會選舉當選國家元首。由於兩者已被魯道夫同時納入手中，國家實際上已沒有人物能抗衡魯道夫政權。因此，在宇宙曆３１０年，魯道夫．馮．高登巴姆篡奪了銀河聯邦政府，登基為帝，是為「銀河帝國．高登巴姆王朝」。

他和皇后所生的四個孩子全是女兒，沒有繼承其位的男孩。晚年，寵妾為他生了一名男嬰，但據傳是先天性的痴呆兒。帝國曆４２年魯道夫大帝結束了長達八十三年的壽命，皇位由長女的兒子繼承。

# 惡劣遺傳因子排除法：

「惡劣遺傳因子排除法」是魯道夫大帝於帝國曆９年所發佈的，其目的在使身體殘障者、貧困無依者和「非優秀」的人完全絕種。讓精神失常的人安樂死，並廢除救濟貧弱的社會政策。此法揭示後引起了議會的共和政治家的批評，魯道夫大帝因此而下令永久解散國會，並於次年在帝國內政部成立「社會秩序維護局」，大力整肅政治犯。此惡法在第二十三代皇帝──「晴眼帝」馬克西米利安．由謝夫二世在位時被有名無實化，但仍要待新皇朝皇帝萊因哈特執政後才正式廢止。

# 宇宙曆：

西元２８０１年銀河聯邦成立時，將西曆改為宇宙曆，宇宙曆正式成為人類社會的標準曆。魯道夫篡奪銀河聯邦政府時，宇宙曆一度停用，直到亞雷．海尼森帶領共和主義者們長征一萬光年後，建立「自由行星同盟」，宇宙曆才重新被採用為標準曆。由於宇宙曆並非由自由行星同盟建國後重新制定，而是重新採用，故宇宙曆仍舊是由銀河聯邦成立時所算起，比帝國曆長三百零九年。

# 帝國曆：

帝國曆有分「舊帝國曆」與「新帝國曆」。「舊帝國曆」是由魯道夫大帝建立銀河帝國時，停用宇宙曆而改用帝國曆時所算起，是高登巴姆王朝的標準曆，比標準的宇宙曆短三百零九年。「新帝國曆」是由新皇帝萊因哈特建立羅嚴克拉姆王朝時正式改用，與標準宇宙曆相差七百九十八年。

# 馬克：

帝國馬克，是銀河帝國的通用貨幣單位。由於費沙自治領名義上附庸於銀河帝國，故其貨幣單位也與宗主國銀河帝國相同，也是「馬克」。而為了區分，費沙自治領的馬克則被喚為「費沙馬克」。

# 帝納：

自由行星同盟的通用貨幣單位。

# 亞雷．海尼森：

銀河帝國為了保持國家的強勢和鞏固皇帝的權威，在帝國曆１０年設立「社會秩序維護局」，大力整肅政治犯，將反對帝制的共和主義者們流放到邊境的星域作奴隸。亞雷．海尼森是一群被流放到牛郎星（天鷹座α）上，被降為奴隸的共和主義者之中的一人。他用天然乾冰做成的太空船離開牛郎星，並帶領共和主義者們隱身於一無名行星的地下，建造了八十艘星際太空船，展開了被稱為「長征一萬光年」的漫漫長路。可惜他未能親眼目睹長征的結局而在旅途中辭世，後人為紀念他而尊他為「國父」，並將首都命名為「海尼森」。

# 長征一萬光年：

銀河帝國為了保持國家的強勢和鞏固皇帝的權威，在帝國曆１０年設立「社會秩序維護局」，大力整肅政治犯，將反對帝制的共和主義者們流放到邊境的星域作奴隸。帝國曆１６４年，被流放到酷寒、有著豐富天然乾冰的牛郎星（天鷹座α）上的青年亞雷．海尼森，在機緣巧合下，撞見一群小孩把冰塊削成小船玩耍。他從中得到靈感，用豐富的天然乾冰做成太空船，帶領共和主義者們離開。潔白晶瑩的乾冰太空船，名為「伊歐．法傑卡斯」，是以製做冰船的少年的名字所命名。四十萬名男女乘著這艘船逃離牛郎星系，為了躲避帝國軍，他們隱身於一無名行星的地下，建造了八十艘星際太空船，然後再航向銀河系未被踏足的地帶。領導人亞雷．海尼森在旅程中逝世，其好友古恩．基姆．霍爾接下了他的棒子。終於，在五十四年後，共和主義者們終於脫離了危險地帶，進入安定的壯年期星群。帝國曆２１８年，時值宇宙曆５２７年，自由行星同盟宣佈成立，並以巴拉特的第四行星為根據地，命名為海尼森以緬懷他的不朽功績。這五十四年間的旅程，被後世稱之為「長征一萬光年」。（詳細可見於《銀英傳》重要史記）

# 巴拉特：

古代腓尼基神祇的名稱。長征一萬光年的共和主義者們，以古代腓尼基諸神的名稱替新天地的星群命名，而巴拉特星域的第四行星成為了共和主義者們的根據地，並以領導者海尼森之名命名。宇宙曆８００年，自由行星同盟正式滅亡，海尼森作為同盟首都的任務結束，成為了新銀河帝國的轄下星球，而巴拉特亦成為新銀河帝國的轄下星域。到了宇宙曆８０１年，伊謝爾倫共和政府與皇帝萊因哈特達成和議，海尼森與其所屬的巴拉特星系成為了共和政治的最後根據地。

# 瓦爾哈拉：

北歐神話中，諸神均住在「亞斯加特」（即天界），主神奧丁在天界中央興建了一座「英靈殿」。日耳曼文化相信，英勇的軍人戰死沙場後，必然會升上天堂的「英靈殿」（Valhalla），亦即「瓦爾哈拉」。銀河帝國便以此命名首都奧丁所在的星系。自建立銀河帝國後建都於此，四百九十年間一直是「銀河帝國．高登巴姆王朝」的首都星系。

# 伊謝爾倫回廊：

夾處於帝國與同盟之間，有著一片充滿變光星、紅色巨星、異常的重力場等危險星體的地帶，而在這密集地帶之中，有兩條細小的安全地帶，一為「伊謝爾倫回廊」，另一則是「費沙回廊」。伊謝爾倫回廊距離帝都奧丁６２００光年，在回廊中有一顆壯年星亞爾提那，是一顆沒有行星的孤獨恆星。

帝國軍於帝國曆４５３年，著手興建直徑６０公里的伊謝爾倫要塞，並於帝國曆４５７年完成。同盟軍前後六次大舉來襲，每一次都因其要塞主炮──雷神之鎚，而大量折損官兵和艦艇。回廊的控制權曾數度易手，宇宙曆７９６年的第七次伊謝爾倫攻略戰，同盟軍楊威利少將成功攻佔了要塞。宇宙曆７９８年的第九次伊謝爾倫攻略戰，帝國軍羅嚴塔爾一級上將成功奪回要塞。其後的第十次伊謝爾倫攻略戰，回廊的控制權又再易手，成為艾爾法西爾獨立政府和伊謝爾倫共和政府的根據地。直到宇宙曆８０１年，伊謝爾倫共和政府與皇帝萊因哈特達成和議，回廊控制權的戰爭才告停止。

# 伊謝爾倫要塞：

伊謝爾倫要塞是帝國軍於帝國曆４５４年，宇宙曆７６３年建造，帝國曆４５８年，宇宙曆７６７年完成。直徑６０公里，內部若加以細分，可以分隔成數千層。表層以耐光束的鏡面處理而成，是由超硬度鋼、結晶纖維及特製陶瓷所組成的複合裝甲，共有四層，牢不可破。外壁表面積達１１３００平方公里，設有許多索敵系統、砲台、槍座及艙口，彼此互相監視，但並非全無死角，敵軍便多利用死角以陸戰進攻。以戰略基地而言，其戰略機能可說是樣樣兼備，攻擊、防禦、補給、保養、裝備、醫療、通訊、管制、情報等，不一而足。宇宙港口可停泊二萬艘艦艇，裝備工廠可以同時修復四百艘戰艦；醫院共可容納二十萬張床位；兵工廠一個小時可以生產７５００枚雷射核融合飛彈。其要塞主炮──雷神之鎚，輸出達９２４兆瓦，同盟軍前後六次大舉來襲，每一次都因此而大量折損官兵和艦艇。

伊謝爾倫既是要塞，同時也是可容納五百萬人口以上的大都市。這裡的社會資本堪稱完備，而且各項設施俱全，從學校、劇場、音樂廳、高達十五層的運動中心、婦產科醫院、育兒所，到內部一體成型的給水排水系統、淡水工廠暨氫氣動力爐、可充作氧氣供給系統之一及森林浴場所的廣大植物園，以及最重要的水耕農場──此為植物性蛋白質和維他命的補給站。

要塞自宇宙曆７６７年完成以來，一直是帝國軍對同盟用兵的要衝，直到宇宙曆７９６年的第七次伊謝爾倫攻略戰，同盟軍楊威利成功攻佔了要塞，伊謝爾倫要塞才更換了主人。帝國軍時代，要塞司令官與駐留艦隊司令官分由二人擔任，一直以來，兩個職位之間一直不和。楊威利能成功不流血地攻佔了要塞，與這矛盾不無關係。要塞落入同盟軍之後，兩個職位便由一人所兼任，而自奪取到要塞重回帝國之手的三年間，「要塞司令官暨駐留艦隊司令官」一職一直由楊威利一人所擔任。到了「銀河帝國．羅嚴克拉姆王朝」，皇帝萊因哈特亦因襲此舉，合併了二職。

# 費沙回廊：

夾處於帝國與同盟之間，有著一片充滿變光星、紅色巨星、異常的重力場等危險星體的地帶，而在這密集地帶之中，有兩條細小的安全地帶，一為「伊謝爾倫回廊」，而另一則是「費沙回廊」。恆星費沙有四個行星，第二行星是費沙回廊唯一有人居住的地方。帝國曆３７３年，出身於地球的費沙商人雷歐波特．拉普在地球資本的協助下，向銀河帝國要求自治，費沙在地球資本和地球教宗教勢力的協助下而成功自治。自此，費沙回廊成為了帝國及同盟之間的都市國家，以經濟力控制銀河系。而回廊在費沙自治期間，一直是停戰地帶，帝國及同盟均不可對其進行任何軍事行動。到了宇宙曆７９８年，在帝國軍遠征同盟的「諸神之黃昏」作戰中，行星費沙被帝國軍佔領，自此費沙回廊被銀河帝國所控制。

# 行星費沙：

恆星費沙有四個行星，只有第二行星的表面有地殼，而氣體成份亦與人類故鄉地球無異，即百分之八十的氮氣和百分之二十的氧氣，故成為了費沙星域唯一有人居住的地方。但由於行星缺乏二氧化碳水份，故而植物沒法在此生存。星球上的水份也相當少，故行星上紅色的部份是岩石沙漠，要靠著藍藻類等高等植物種子的散佈，使得星球可綠化、沃土。而飲食用水也得要由外輸入。

由於行星費沙的貿易非常繁忙，每天出港的宇宙船不計其數，為了保護行星費沙的自然環境，避免頻繁的宇宙船出入港污染和破壞環境，故行星費沙的宇宙港是建設於衛星軌道之上，利用真空的吸管作為昇降機，以連接宇宙港與行星地面。行星費沙在地球資本和地球教宗教勢力協助下，成功建立自治領，以經濟力控制銀河系。在宇宙曆７９８年之前，費沙一直是停戰地帶，帝國及同盟均不可對其進行任何軍事行動。直到「諸神之黃昏」作戰後，行星費沙被帝國軍佔領，名為自治，但已成銀河帝國的直轄星球。新皇帝萊因哈特篡位後遷都往費沙，「帝都費沙」自此成為了新銀河帝國和人類世界的核心。

# 自由行星同盟最高評議會：

自由行星同盟最高評議會是由十一名評議委員所構成。由議長、副議長兼國務委員長、書記、國防委員長、財政委員長、天然資源委員長、人力資源委員長、經濟開發委員長、地域社會開發委員長和情報交通委員長所組成。最高評議會是自由行星同盟民選的最高權力機關。

# 帝國宰相：

除了銀河帝國皇帝之外，帝國宰相便是銀河帝國最高的權力者。自從第三十一代皇帝奧特佛利特三世在皇太子的時候兼任帝國宰相及帝國三長官後，為了避諱，後世的帝國宰相均非正式，只由內閣中的首席──國務尚書所代理；而帝國三長官則不可一人兼任。直至萊因哈特掌握了銀河帝國的獨裁實權後，這不成文的規條才被打破。帝國宰相的職務是統御內閣，所有尚書均直屬帝國宰相，而其職位及權責是由銀河帝國皇帝所授予。

# 帝國軍三長官：

帝國軍三長官是銀河帝國軍部中最高的三大職位，分別是軍務尚書、統帥本部長及宇宙艦隊司令官。自從第三十一代皇帝奧特佛利特三世在皇太子的時候兼任三長官後，為了避諱，帝國三長官不可一人兼任。直至萊因哈特掌握了實權後，規條才被打破。

# 軍務省：

軍務省是銀河帝國軍中最高的機關，帝國軍興建一切軍事硬體設施，以及帝國軍部的人事，均需軍務省首長軍務尚書所批核。軍務省是內閣之一，其地位與職能與同盟評議會中的國防委員會相近，是帝國軍部中最高的政治機關。軍務尚書是銀河帝國軍部中最高的職位，其職位及權責是由銀河帝國皇帝所授予。

# 國防委員會：

國防委員會是自由行星同盟最高評議會中專職軍事的政治機關，首長是國防委員長。同盟軍興建一切軍事硬體設施，以及同盟軍部的人事，均需國防委員會所批核。國防委員會的職能與帝國的軍務省相近，最大的不同是，軍務省是帝國軍部中最高的機關，軍務尚書是由帝國軍的軍官來擔任；國防委員會則是民選的政治機關，國防委員長是由民選的政治家來擔任。

# 統合作戰本部：

統合作戰本部是同盟軍軍部中最高的機關，其職位及權責由自由行星同盟最高評議會所授予。同盟軍一切戰略立案均由統合作戰本部所批核。首長是統合作戰本部長，是同盟軍官中最高的職位，職能與帝國軍統帥本部相近。

# 統帥本部：

統帥本部是帝國軍中僅次於軍務省的機關，首長是統帥本部長，是軍官中第二高的職位。統帥本部的職能及權責是由銀河帝國皇帝所授予。帝國軍一切軟體如戰略立案等，均由統帥本部所批核。其職能與同盟軍的統合作戰本部相近，是皇帝的參謀總部。

# 宇宙艦隊：

宇宙艦隊是同盟軍與帝國軍兩方，在宇宙戰場戰鬥的部隊。宇宙艦隊司令官是兩國宇宙艦隊的最高指揮官，亦是兩國軍部實際作戰的最高指揮（法律上兩軍的最高指揮官是帝國皇帝與最高評議會議長）。兩軍的宇宙艦隊編制和實際作戰的指揮，均由宇宙艦隊司令官所批核和執行。宇宙艦隊司令官是帝國軍部排行第三的職位；同盟軍中則是軍部排行第二的職位。

# 元帥府：

設立元帥府是銀河帝國元帥才有的特權，可說是一種派閥組織。元帥可自由將優秀的人材納入自己麾下，不受軍部控制，可以作為帝國元帥專屬的參謀組織。而帝國元帥更擁有除謀反等大不敬罪外，其餘一切罪行均可免責的特權。

# 同盟軍最高幕僚會議：

自由行星同盟軍中特設軍方幕僚組織，是直隸於統合作戰本部的參謀組織，並無實權。「亞姆立札星域會戰」後，楊威利也是議員之一，不過由於那只屬咨詢機關，所以並無實質職權。

# 幕僚總監：

新帝國曆２年，羅嚴克拉姆王朝元帥羅嚴塔爾被誣陷反逆，皇帝萊因哈特與羅嚴塔爾冰釋後，解除了羅嚴塔爾統帥本部長的職務，並任命為新領土總督。而統帥本部則由皇帝萊因哈特親自統轄，設立幕僚總監來輔助，第一任幕僚總監由舒坦梅茲一級上將所擔任。後來舒坦梅茲在回廊會戰中戰死，第二任幕僚總監則由當時尚未成為皇妃的宰相秘書官希爾格爾所擔任。

# 社會秩序維護局：

魯道夫大帝於帝國曆９年發佈惡劣遺傳因子排除法，此法揭示後引起了議會的共和政治家批評，魯道夫因此下令永久解散國會，並於次年（帝國曆１０年）在帝國內政部成立「社會秩序維護局」，大力整肅政治犯。魯道夫任命心腹──內政大臣亞倫斯特．法斯特隆為局長，在「法律無效，主觀判斷至上」的前提下，展開逮捕、拘禁、下獄及懲罰行動。由於維護局的手段相當殘酷，因此有「不想被判死刑就去讓警察逮到。被社會秩序維護局抓到便只有死路一條」的說法。

社會秩序維護局在「銀河帝國．高登巴姆王朝」的四百九十年間，一直是支配民眾、壓制思想的政策實行機關，直到帝國曆４８９年，羅嚴克拉姆公爵掌權才予以廢除。

# 救國軍事會議：

救國軍事會議是宇宙曆７９７年，於自由行星同盟首都海尼森發動政變的組織，宣言中止同盟憲章，以打倒銀河帝國為崇高目標，希望以軍事獨裁的方式拯救同盟打倒帝國。事緣於該年２月，同盟政府接受了銀河帝國的交涉，同意交換俘虜。但交換的同盟俘虜中，包括了間諜林奇少將，依循帝國軍所設計的計劃分裂同盟軍，以阻止同盟介入帝國的內戰。

宇宙曆７９７年４月１３日，同盟首都海尼森發生政變。統合作戰本部代理部長德森與宇宙艦隊司令官比克古遭到監禁，元首特留尼西特失蹤。政變組織自稱「救國軍事會議」，德懷特‧格林希爾上將擔任議長，並派遣魯格拉希與楊上將決戰。楊艦隊擊破第十一艦隊，魯格拉希自殺。

６月２２日，發生了「海尼森紀念廣場大屠殺」事件，民意堅拒軍事支配。楊艦隊在８月到達巴拉特星域，以逾千億噸質量的冰塊毀滅了「處女神之首飾」。革命軍內部產生強烈動搖。格林希爾被殺，代理議長艾凡思上校宣佈投降，議會解散，歷時四個月政變終結。（詳細可見於《銀英傳》重要史記及重要戰史）

# 利普休達特的密約：

帝國曆４８７年１０月，第三十六代皇帝佛瑞德李希四世猝逝。由於皇帝尚未指定繼承人便死了，而佛瑞德李希的兒子都早死，沒有嫡子，國務尚書立典拉德侯爵便銳意扶立五歲的皇太孫──艾爾威．由謝夫繼承皇位。為抗衡佛瑞德李希四世的兩位女婿──歐特．布朗胥百克公爵和威爾罕姆．立典亥姆侯爵，國務尚書便拉攏了從亞姆力札星域凱旋歸來，沒有貴族勢力的萊因哈特，二人合掌實權。布朗胥百克公爵和立典亥姆侯爵對此非常不滿，他倆均希望由二人各自的女兒登上皇位，以攝政王的身份把政，藉以掌握實權。因此銀河帝國的統治階級分裂為兩派：一是擁護皇帝派，以立典拉德與萊因哈特為樞紐；另一為反皇帝派，由布朗胥百克與立典亥姆連手主陣。帝國曆４８８年２月，反對羅嚴克拉姆侯爵與立典拉德公爵陣營的貴族們，紛紛前往位於奧丁的布朗胥百克公爵別墅所在的利普休達特森林集合。集合的名義是古代名畫的拍賣會與園遊會，而實際則在地下的大廳之中，進行著反對羅嚴克拉姆侯爵與立典拉德公爵「專橫政治」的「愛國簽名」活動。

這項運動通稱「利普休達特盟約」，根據盟約所建立的貴族軍事組織，便稱為「利普休達特反帝聯合軍」。盟主是歐特．布朗胥百克公爵，副盟主是威爾罕姆．立典亥姆侯爵。參加的貴族有３７４０名，正規軍及私人專屬士兵，兵力共計達２５６０萬。近４０００個貴族的名字所串連的署名狀，以激烈的論調，抨擊立典拉德與羅嚴克拉姆。另外，還高高地揭櫫守護高登巴姆王朝的神聖使命，己賦予在「天選之民」的傳統貴族階級。並以「大神奧丁守護著我們！正義的勝利非我們莫屬！」為署名狀全文的結尾。

# 貴族聯合軍：

帝國曆４８７年，第三十六代皇帝佛瑞德李希四世猝逝。國務尚書立典拉德侯爵扶立皇太孫艾爾威．由謝夫繼承皇位。身為先帝女婿的布朗胥百克公爵和立典亥姆侯爵不滿，他倆均希望自己女兒登上皇位，以攝政王的身份掌握實權。帝國曆４８８年２月，反對羅嚴克拉姆侯爵與立典拉德公爵的貴族，在布朗胥百克別墅的利普休達特森林結盟，運動通稱「利普休達特盟約」。根據盟約所建立的貴族軍事組織，便稱為「利普休達特反帝聯合軍」，亦即「貴族聯合軍」，而擁護皇帝的正規帝國軍則以「賊軍」稱呼。貴族聯合軍集結在禿鷹之城要塞，並任命維利伯爾．由希姆．馮．梅爾卡茲一級上將為實戰總司令，參與的提督還有斯特汀上將與法倫海特中將等。貴族聯合軍與中央政府軍在４月展開第一次交戰，政府軍首捷。６月，副盟主立典亥姆在奇霍伊薩星域陣亡。節節敗退的貴族軍士氣低落，直到８月，貴族聯合軍從據守的禿鷹之城要塞出戰被政府軍所殲滅。由於發生了「威斯塔朗特事件」，貴族軍的平民士兵群起叛變，貴族軍潰滅。

梅爾卡茲一級上將聽取副官舒奈德少校所勸逃亡到自由行星同盟，投靠楊威利。而布朗胥百克公爵則被餵毒酒自殺。安森巴哈藉面見俘虜的機會暗殺萊因哈特失敗，卻殺害了吉爾菲艾斯一級上將，安森巴哈自殺身亡。歷時接近一年的內戰正式完結。（詳細可見於《銀英傳》重要史記及重要戰史）

# 禿鷹之城要塞：

帝國軍要塞禿鷹之城，質量在４０兆噸以上，直徑介於４０至４５公里，可容納１６０００艘船艦及二百萬名官兵，材質為合金及陶瓷複合材，擁有結合超硬度鋼、結晶纖維和超級陶瓷的四度複合裝甲，功能、素材、戰力等與伊謝爾倫要塞相近，但規模比伊謝爾倫要塞較小。要塞主炮是X 射線光束砲，又有稱為「禿鷹之啄」，光束波長１００埃，輸出達７４０兆瓦。

禿鷹之城要塞在帝國曆４８８年的利普休達特戰役中，被貴族聯合軍作為基地而知名，而名將吉爾菲艾斯便是在此身亡。由於貴族軍並沒有善用要塞戰力，故直到帝國曆４８９年的第八次伊謝爾倫攻略戰，禿鷹之城要塞才被作為對抗同盟軍的重要戰力。帝國曆４８９年１月，帝國科學技術總監安東．希爾曼．馮．胥夫特技術上將，提出在禿鷹之城要塞安裝瓦普跳躍引擎，然後運送到伊謝爾倫回廊，以對抗同盟軍的伊謝爾倫要塞的計劃。最高司令官羅嚴克拉姆任命坎普上將實行計劃，安裝了２４個瓦普引擎的要塞在３月１７日實驗飛行成功，帝國軍隨即展開遠征。禿鷹之城要塞在４月１０日到達伊謝爾倫回廊，與同盟軍展開了為時４週的攻防戰。５月，被召回首都海尼森的楊威利上將趕到戰場，坎普艦隊在楊上將援軍與伊謝爾倫駐留艦隊的夾擊下潰滅，禿鷹之城要塞被楊艦隊攻勢下，要塞因核融合爐爆炸而毀滅，坎普在爆炸中陣亡。（詳細可見於《銀英傳》重要戰史）

# 連典貝爾克要塞：

連典貝爾克要塞位於帝國弗列亞星系裡的一個小行星上，雖比不上伊謝爾倫要塞與禿鷹之城要塞的龐大，但也可容納一百萬個將兵和一萬艘以上的艦艇。內部還包括了戰鬥、通信、補給、整備，以及醫療等各方面的功能，戰略上相當重要。

連典貝爾克要塞在帝國曆４８８年的內戰時是貴族聯合軍所設定的第三處據點，正規帝國軍因其戰略性而展開攻擊。由於裝甲擲彈兵總監奧夫雷沙一級上將的頑強抵抗，令進攻的帝國正規軍傷亡慘重。後來帝國軍製造陷阱活捉了奧夫雷沙，當時羅嚴克拉姆元帥的參謀長奧貝斯坦設計放回奧夫雷沙，讓多疑的盟主布朗胥百克公爵懷疑奧夫雷沙背叛，並予以殺害。此反間計擴大了貴族們互不信任的弱點，令貴族們人人自危，最後間接加速了貴族聯合軍的分裂和敗亡。

# 薔薇騎士連隊：

「薔薇騎士」連隊是同盟軍陸戰總監部所屬的地上作戰部隊，以帝國流亡至同盟的貴族子弟為中心所創設，約有半個世紀的歷史。在其歷史中有過金碧輝煌的部份，卻也有過被塗污抹黑的部份。歷代隊長共有十五名，五名在戰鬥中死亡，三名出任將官之後退位，一名在戰爭結束後退役，六名則逃回舊祖國──有的是秘密脫逃的，有的是在戰鬥中認敵為友的。

在宇宙曆７９６年到宇宙曆８０１年間，薔薇騎士連隊在與銀河帝國不斷的戰鬥中，人數逐漸減少。到了宇宙曆８０１年，伊謝爾倫共和政府與帝國軍交戰時，其人數甚至不足一個中隊。在這段期間，薔薇騎士連隊參與了第七、第八、第九及第十次伊謝爾倫攻略戰、香普爾星域解放作戰、海尼森市街戰、「血之階梯瀑布」等多場戰役。薔薇騎士之勇猛是全同盟軍之首，素有「同數的兵力下，薔薇騎士無人能敵」的說法，這說法在敵我雙方的軍隊中都產生一定的心理影響。

# 裝甲擲彈兵：

裝甲擲彈兵是銀河帝國軍的地上作戰部隊，部隊與工兵隊等均屬獨立部隊，不直屬於任何獨立艦隊，直接隸屬於統帥本部。部隊首長為裝甲擲彈兵總監，而其中最有名的便是在「利普休達特戰役」中被殺的奧夫雷沙一級上將，職位與各獨立艦隊的司令官相當。

# 王爾古雷：

王爾古雷是帝國軍的Ｘ型機翼單座式戰鬥艇，其名字出於北歐神話，以北歐神話中的女武神（Walkure）命名。王爾古雷機型小巧，火力較同盟軍的斯巴達尼恩差，但活動性強，最適合作近距離肉搏戰。

# 斯巴達尼恩：

斯巴達尼恩是同盟軍的單座式戰鬥艇，機師要利用一張記錄著其姓名、血型、指紋、聲音、DNA型態、軍藉號碼、階級等資料的ID卡，經電腦分析核實資料後，機師才能開動駕駛。頭盔有二道密碼與電腦直接聯繫，傳達駕駛員腦電波，如果腦波與電腦記憶中的記錄有異，頭盔會發射低輸出的高壓電擊，立即致人昏迷。斯巴達尼恩機型較王爾古雷大，而且活動性較差，但火力較強，是艦隊近距離戰的必要武器。

# 雷神之鎚：

伊謝爾倫要塞的要塞主炮，輸出率達９２４兆瓦。在舊帝國時代，同盟軍前後六次大舉來襲，每一次都因「雷神之鎚」而大量折損官兵和艦艇。其名字亦是出於北歐神話，以北歐神話中的雷神──托爾的鐵鎚命名。北歐神話中，相傳雷神托爾是神通廣大的戰神，加上他有一柄威力無窮的鐵鎚，令魔界人人膽顫心驚。帝國軍便以托爾威力無窮的鐵鎚來比喻伊謝爾倫要塞主炮的強勁破壞力。

# 處女神之首飾：

「處女神之首飾」是設置在自由行星同盟首都海尼森的衛星軌道上，共十二個的迎擊衛星。迎擊衛星具有三百六十度、全方向的攻擊能力，雷射炮、荷電粒子光束炮、中子光束炮、熱線砲、雷射氫爆飛彈、磁力炮等，武器裝備應有盡有，利用太陽能可半永久性地補給動力，擁有準完全鏡面的裝甲。在衛星軌道上自由移動的十二個衛星，具備相互防禦、支援的機能。因此，即使同時對幾個衛星攻擊，必得承擔分散戰力的危險。以銀色為中心，四周閃耀霓虹光彩，是一個近乎高貴般美麗的大量殺人系統。雖然其威力強大，但自由行星同盟史上卻從未被使用過，最後在宇宙曆７９７年被楊威利上將以冰塊所摧毀。（詳細可見於《銀英傳》重要戰史）

# 禿鷹之啄：

禿鷹之城的要塞主炮，是X射線光束砲，光束波長１００埃，輸出率達７４０兆瓦，規模略遜伊謝爾倫主砲雷神之鎚一籌。

# 達貢殲滅戰：

宇宙曆６４０年，帝國曆３３１年２月，銀河帝國與自由行星同盟發生了第一次軍事接觸，同盟軍大獲全勝。同年７月，銀河帝國組織了討伐軍，以三王子海貝爾特為討伐軍的總司令，遠征同盟。而同盟議長與國防委員長，則任命林．帕歐中將與尤斯夫．托波洛中將為迎擊部隊的總司令及參謀長。

帝國軍輕視此次遠征，在補給與情報不明的情況下輕率進攻，種下了兵敗禍因。同盟軍迎擊部隊，將不諳地形、輕敵的討伐軍引至地勢複雜的達貢星域。由於同盟軍熟識地勢複雜的達貢星域，帝國軍面對同盟軍的遊擊戰束手無策，無謀的海貝爾特下令分散部隊搜索，令帝國軍出現兵力分散的弱點。同盟軍將部隊分成三路包圍討伐軍司令部，上演了一場包圍殲滅戰。由於帝國軍兵力分散，同盟軍能成功完成包圍網，殲滅了兵力遠比自軍強大的帝國軍，總司令海貝爾特倉皇逃出歸國。

帝國軍總司令海貝爾特歸國後因兵敗而失去了成為皇帝的機會，被軟禁於離宮。而參謀英格爾休塔中將被代罪為敗戰的禍首，被判死刑。同盟軍方面，總司令林．帕歐中將與參謀長尤斯夫．托波洛中將被晉升為元帥，成為了同盟軍史上的名將而名垂千古。（詳細可見於《銀英傳》重要戰史）

# 伊謝爾倫攻略戰：

歷史上共展開了十次伊謝爾倫攻略戰，其中八次是由同盟軍進攻，二次是由帝國軍進攻。伊謝爾倫要塞位於伊謝爾倫回廊中心，是二國的唯一交界。由於費沙回廊是停戰地帶，所以二國的交戰均是圍繞伊謝爾倫回廊與及其周邊星域所發生。要塞由帝國軍於帝國曆４５７年，宇宙曆７６７年完成，一直是帝國軍對同盟軍進攻及防禦的橋頭堡，頭六次的攻略戰均令同盟軍不斷損兵折將，直到宇宙曆７９６年的第七次伊謝爾倫攻略戰，楊威利以「不流血」的方式成功攻佔了要塞，兩者的形勢才告逆轉。直到第九次伊謝爾倫攻略戰，伊謝爾倫要塞才重新被帝國軍佔據。而第十次伊謝爾倫攻略戰，要塞的支配權因楊威利的奇謀而再度易手。頭四次的攻略戰並無記述。（詳細可見於《銀英傳》重要戰史）

# 迪亞馬特星域會戰：

迪亞馬特星域是同盟境內，鄰近伊謝爾倫回廊的邊境星域。自從達貢殲滅戰兩軍正式交戰以來，同盟軍共在這兒展開了四次對帝國軍的迎擊作戰。由於費沙回廊是停戰地帶，所以二國的交戰均圍繞伊謝爾倫回廊與及其周邊星域所發生。第一次迪亞馬特星域會戰並無記述。宇宙曆７４５年，帝國曆４３６年的第二次迪亞馬特星域會戰則是同盟軍史上另一次大捷，迎擊部隊的總司令正是同盟軍史上的名將──布魯斯．阿修比上將。宇宙曆７９５年，帝國曆４８６年２月的第三次會戰，雙方則在互無重大勝負的情況下退兵。同年９月的第四次迪亞馬特星域會戰，帝國軍得勝，萊因哈特．馮．繆傑爾上將在此役中功績超卓，受封為羅嚴克拉姆伯爵，自此平民和下級貴族出身的帝國軍官，勢力抬頭的情況更益顯著。

雖說帝國軍曾在此星域得勝，但由於戰役並非以長期佔據為前題展開，所以迪亞馬特星域並無長期成為帝國領域的歷史。（詳細可見於《銀英傳》重要戰史）

# 利普休達特戰役：

帝國曆４８７年，第三十六代皇帝佛瑞德李希四世駕崩。國務尚書立典拉德扶立皇太孫，艾爾威．由謝夫二世繼承皇位，並邀請羅嚴克拉姆共同執政。身為先帝女婿的布朗胥百克公爵和立典亥姆侯爵非常不滿，在帝國曆４８８年２月，反對羅嚴克拉姆與立典拉德的貴族，在布朗胥百克公爵別墅的利普休達特森林結盟，被通為「利普休達特盟約」，而整個貴族聯合軍的叛亂被稱為「利普休達特戰役」，而其中又專指貴族軍在禿鷹之城要塞外與中央政府軍決戰的一役。（詳細可見於《銀英傳》重要史記及重要戰史）

# 諸神之黃昏：

帝國曆４９８年、宇宙曆７９８年，費沙領主魯戴斯基為了挑撥銀河帝國與自由行星同盟之間的戰事，策動陰謀以舊貴族勢力，誘拐了銀河帝國幼帝艾爾威．由謝夫二世，並逃亡到敵國自由行星同盟。帝國宰相萊因哈特．馮．羅嚴克拉姆，以此事作為興兵遠征自由行星同盟的大義理由，遠征行動是為「諸神之黃昏」作戰。

「諸神之黃昏」作戰是一個極大規模的遠征計劃，歷時由帝國曆４９８年到４９９年（宇宙曆７９８年到７９９年）。計劃第一部份是由羅嚴塔爾一級上將率兵進攻伊謝爾倫回廊，一方面是以奪回要塞為目標，同時也負責牽制位於伊謝爾倫的楊威利艦隊。第二部份由米達麥亞一級上將率領艦隊以快攻攻佔行星費沙，以控制能代替伊謝爾倫作為進攻路線的費沙回廊。帝國曆４９９年初，羅嚴塔爾在楊艦隊主動撤離下奪回伊謝爾倫要塞，而費沙方向的帝國軍主力則經費沙回廊進攻同盟領，並在蘭提馬利歐星域與同盟軍交戰獲勝。

帝國軍在干達爾星域集結，但因為楊艦隊的游擊戰而開始缺乏物資，帝國軍三位上將都成為了楊艦隊的手下敗將。帝國軍為了引出並消滅楊艦隊，主帥萊因哈特設計出大規模的包圍殲滅戰。４月４日開始，帝國軍各艦隊陸續從根據地干達爾星域出發，攻擊各個同盟星域的基地，並適時回轉以包圍殲滅被引出的楊艦隊。楊艦隊與萊因哈特直屬艦隊在巴米利恩星域交戰，萊因哈特雖得及時回到戰場的繆拉支援，但還是節節敗退。最後因為希爾格爾聯合米達麥亞與羅嚴塔爾艦隊，包圍同盟首都海尼森，並脅逼同盟政府投降後，同盟政府下令楊艦隊停止攻擊，萊因哈特才得以生還。

最後在帝國曆４９９年（新帝國曆１年）５月２５日，自由行星同盟在城下之盟的情況下，與銀河帝國簽訂了「巴拉特和約」，同盟得保了內政自治權和象徵性的軍隊，但與成為帝國附庸國無異。帝國軍凱旋回國，第一次的「諸神之黃昏」作戰結束。

新帝國曆１年，因為同盟政府對高等事務官雷內肯普，要求拘捕楊威利退役元帥的問題處理不當，帝國駐同盟高等事務官雷內肯普因此而身亡。新帝國政府以此作為理由再次對同盟用兵，是為第二次「諸神之黃昏」作戰。

新帝國曆２年，帝國軍在馬爾．亞迪特星域與同盟軍交戰得勝，但同一時間卻因為楊艦隊的奇襲而再次失陷伊謝爾倫要塞。帝國軍在馬爾．亞迪特星域得勝後到達同盟首都海尼森，處死了暗殺同盟元首姜．列貝羅以取悅帝國的同盟軍官後，於新帝國曆２年２月２０日，在海尼森國立美術館內公佈了「冬薔薇園敕令」，正式宣佈同盟政府的滅亡，與及以外交立場承認同盟政府的存在。第二次「諸神之黃昏」作戰，隨著同盟的滅亡而正式結束。（詳細可見於《銀英傳》重要史記及重要戰史）

# 宇宙曆七九八年共濟協定：

被後世稱為「宇宙曆七九八年共濟協定」，是在宇宙曆７９８年８月２０日公開的一項協定。所謂的「共濟協定」，是銀河帝國舊體制派與自由行星同盟之間，相對於羅嚴克拉姆獨裁體系，所相互訂定的協助關係。自由行星同盟承認了艾爾威．由謝夫二世的亡命行動，而且承認了以瑞姆夏德伯爵為首的閣僚班底的流亡政權。

# 審查會：

審查會在同盟憲章和同盟基本法中並沒規定，可說沒有法律根據，超出了法律明文規定。而且也不像正式的軍法會議般，審查會不需要發出正式通告，被害也沒權利申請辯護人，也毋須留下正式記錄，可謂是隨當權者之意所至。有正式記載的審查會可說並不存在，所以沒法統計同盟二百餘年間，曾經召開多少次審查會或同類的非正式軍事法庭。

而最後的一次審查會，相信該是宇宙曆７９８年特留尼西特政權的同盟政府，對當時是戍衛邊境伊謝爾倫要塞的楊威利上將，所進行的「品格與忠誠心」的審查。由於楊威利上將奉命秘密回首都接受審查，所以國民並不知情。加上楊上將對審查並不合作，因此審查會進行並不順利。而此時帝國軍竟大舉來犯，楊上將被召回首都接受審查的事實被外界所知悉，國防委員長尼古拉龐提因此而要負上責任辭職，而盛傳的幕後黑手，議長特留尼西特則繼續手握政權，並任命華爾特．愛朗茲為新任國防委員長。而此次審查會突出了同盟後期的政治狀況，當權者肆意弄權的情況相當嚴重。

# 巴拉特和約：

宇宙曆７９８年，費沙為了挑撥帝國與同盟的戰事，策動舊貴族勢力誘拐了帝國幼帝艾爾威．由謝夫二世，並逃亡到同盟，成立流亡政權。帝國宰相萊因哈特．馮．羅嚴克拉姆，以此事作為興兵遠征同盟的大義理由，是為「諸神之黃昏」作戰。

同盟軍在蘭提馬利歐星域戰敗後，楊艦隊雖然力挽狂瀾，與萊因哈特直屬艦隊在巴米利恩星域交戰而接近勝利，但因為帝國軍包圍同盟首都海尼森，脅逼同盟政府投降後，楊艦隊被令停止攻擊。最後在帝國曆４９９年（新帝國曆１年）５月２５日，自由行星同盟在城下之盟的情況下，與銀河帝國簽訂了「巴拉特和約」，內容如下：

銀河帝國同意自由行星同盟繼續保有名稱與主權。
同盟國把干達爾星系與位於兩回廊出口周邊的兩個星系割讓給帝國。
同盟同意帝國的軍艦及民間船隻在同盟領土之內自由航行。
同盟每年支付帝國１５兆５０００億帝國馬克做為完全保障稅。
同盟得以有象徵主權的軍備，但必須放棄保有戰艦及宇宙母艦的權利。此外，同盟在建設、修改軍事設施之前，必須和帝國取得協議。
同盟制定國內法，禁止以妨礙和帝國友好及協調為目的的活動。
帝國在同盟首都海尼森設置高等事務官府，並享有駐留警備軍隊的權利。高等事務官代理帝國主權者（皇帝）和同盟政府折衝、協議，並且具有旁聽同盟政府各項會議的資格。

第８條以下的這些條文讓雙方確認了同盟已隸屬帝國領土的事實。同盟元首優布．特留尼西特在帝國軍包圍下簽名、蓋章以確認同意，然後宣稱擔負起戰敗責任而引咎辭職，其政敵姜．列貝羅上任代理元首職務。而帝國軍則凱旋回歸，宰相萊因哈特．馮．羅嚴克拉姆正式篡奪了高登巴姆王朝，開創了「羅嚴克拉姆王朝」。

# 冬薔薇園敕令：

新帝國曆２年，宇宙曆８００年２月２０日，新皇帝萊因哈特在自由行星同盟首都海尼森的國立美術館公布的敕令，正式名稱是「新帝國曆２年２月２０日的敕令」，因條文是在國立美術館的冬薔薇園公布的，故又通稱「冬薔薇園敕令」。條文宣佈了自由行星同盟正式滅亡，並首次公開以帝國政府身份宣佈承認同盟政府的存在。

# 托爾巴斯法：

托爾巴斯法是同盟的法案，此法案之正確名稱是「軍人子女戰時福利特別法」，也有稱為「戰時托孤法」。由於訂立這項法案的人名叫托爾巴斯，故此法案又通稱「托爾巴斯法」。帝國與同盟之間已交戰一百五十多年，許多人因戰爭而死亡、傷殘，「托爾巴斯法」就是為了救濟這些失去親人的孤兒，達到確保人力資源的目的，應運而生的法令。這些孤兒們交由軍人家庭來撫養，政府貸給他們一定數額的教養費，孤兒們在十五歲以前一律進一般學校上課，十五歲以後則憑個人的意願選擇自想進的學校就讀。不過，若選擇進軍官學校或技術學校等相關學校的話，則向政府借貸的教養費即可不必歸還。另外，雖然女性無法對軍事行動有所貢獻，但是也很歡迎他們能投入補給、經理、運輸、通信、管制、情報處理及設施管理等行列。

# 三美神：

「三美神」是在人類還生活於太陽系時期的科學成就，使人類能得以在恆星來往飛行，分別是──第二空間跳躍航行法、重力控制技術、慣性控制技術，三項技術被通稱為「三美神系統」。技術發展日新月異，不斷進步，人類對宇宙的對外開拓便由此開始。

# 瓦普跳躍航行法：

西曆２３６０年，人類實現了超光速航行，亦即「瓦普跳躍航行」。當時地球掌政的是地球統一政府，「瓦普跳躍航行」技術便是由地球統一政府的宇宙省中，以安特涅爾．亞諾修博士為首的技術群開發完成。初期的瓦普跳躍飛行，距離非常短，而且對於人體，特別是女性的生育能力有著明顯的不良影響。但到了２３９１年，此項航行技術便在不斷的努力之後達到了完全的實用化，勘察的領域也隨之擴大。到了２４０２年，在卡那普斯星系裡發現了可住行星，星際移民時代於焉展開。

# 密倉睡眠：

密倉床具就是在輕型塑膠製的密閉水倉內放約三十公分滿的濃鹽水，水溫保持３２℃。躺在裡面，與外界的色彩、光熱、音響完全隔離，靜謐舒適。在裡面泡上一個小時，可以恢復身心疲勞，效果相當於熟睡八小時。要在短時間內恢復士兵們因戰鬥而消耗的體力和精力，這無異是最好的方法。

# 傑服粒子：

傑服粒子是以發明者卡爾．傑服的名字來命名的一種化學物質。身為化學家的傑服，為了行星規模的礦物採掘或土木工事，發明了傑服粒子。此種物質的主要特性是可在一定量以上的熱量或能量中產生反應，而在可以控制的範圍內引火爆炸的一種擬似瓦斯。後來，此種技術成為了軍事技術，除作為爆破等方式來使用外，亦被肉搏戰部隊作為逼使敵軍不能使用熱能武器（因使用熱能武器會使傑服粒子爆炸，而令雙方同歸於盡）的戰術武器來使用。帝國曆４８７年，具有指向性的傑服粒子被帝國軍所開發成功，交戰時屢建奇功，多被用來對付敵軍的機雷群。

# 飛行球：

飛行球是一種在重力被固定為０．１５Ｇ的球場內所進行的球賽，是一種把球投入沿著壁面做不規則高速移動的籃框中的單純競賽。雖說是單純競賽，但在空中奪球，或是緩慢回轉控球的姿態，似乎像是一種舞蹈，並隨著選手的個性而有優美或爆發性的表現，是一項相當受喜愛的運動。

# 憂國騎士團：

憂國騎士團是一個激進派的國家主義團體，他們高喊「打倒反國家、反戰爭」的言行而受到注目。據傳是由優布．特留尼西特在幕後所操縱，是極端的主戰派組織。由於在背後有幕後勢力所支持，所以行為相當肆無忌憚，涉嫌與多宗反戰派被偷襲毒打的暴力事件有關。由於幕後勢力的干涉，案件都被刻意忽視，不了了之。到後來警方更多次在憂國騎士團離開後才到達，而被打的反戰派則反被警方控告毆鬥，憂國騎士團卻被讚揚為「愛國組織」，沒被問責。在同盟內戰後，有不少憂國騎士團成員都轉而加入地球教，原因有傳是理念相同，也有傳是因為幕後勢力相同。

同盟滅亡後，主戰的憂國騎士團都沉靜下來。後來在新帝國曆２年３月１日，皇帝萊因哈特在遠征伊謝爾倫前停留海尼森時，海尼森市內發生大爆炸，引起大火，後來責任被推到憂國騎士團之上，憂國騎士團最後被剿滅。

# 地球教：

一個奉地球為神靈的宗教組織。自費沙自治領成立之後，地球教就控制著費沙，企圖依靠費沙來統治宇宙。他們給教徒服用塞奧奇辛以控制教徒，從事恐怖事件。邱梅爾事件後，帝國軍對地球教的總部進行了攻擊，使地球教實力大損并失去了根據地。但地球教殘餘仍制造了暗殺楊威利、棺館大火事件等恐怖事件。宇宙曆 ８０１年，新帝國曆 ３年，地球教殘余被奧貝斯坦全部誘出，遭到徹底消滅。

# 愛德華委員會：

紀念海尼森紀念廣場屠殺事件中死去的潔西卡．愛德華的反戰組織。

# 楊艦隊：

宇宙曆７９６年，楊威利少將被任命為第十三艦隊司令官，艦隊以在亞斯堤會戰中生還的第四及六艦隊殘兵為骨幹，再加上新兵所組成，而這就是楊艦隊的前身。在亞姆力札星域會戰後，楊威利晉升上將，並被派駐到伊謝爾倫要塞防衛，而第十三艦隊合併了亞姆力札會戰後殘存的第十艦隊戰力，組成為伊謝爾倫駐留艦隊，一般通稱為「楊艦隊」。但這在同盟正式公文上仍舊稱為第十三艦隊，而這情況在艾爾．法西爾獨立政府和伊謝爾倫共和政府的艦隊編制中亦是相近。反而，帝國軍對於楊艦隊，較同盟艦隊的存在更為重視。由於舊帝國軍的編制是近於私設軍隊的編制，是以「司令官」為核心，與同盟以「艦隊編號」為核心的編制不同（即「兵有常帥」與否的分別），所以將「楊艦隊」或「楊威利軍」等名稱作為公式稱呼的情況，反而在帝國軍中更為普遍。

由於情況特殊，在帝國軍「諸神之黃昏作戰」其間，楊艦隊的編制相當混亂，在艦數和兵數等方面，已脫出同盟軍宇宙艦隊的正常編制。在不斷吸收在蘭堤馬利歐會戰中的殘兵，與及同盟全國各地的義勇軍的情形下，在巴米利恩會戰時，楊艦隊的艦數為１６４２０艘，約比正常編制多出二至三成。而其後到了回廊會戰時，艦隊規模更昇至２８８４０艘，是正常編制的兩倍以上。楊威利身亡後，楊艦隊的編制隨著回廊會戰的激戰，和有部份人對前途不樂觀而離去後，艦艇數由回廊會戰前的２８８４０艘，下降至只有６６００艘，兵員數也只有約６０萬，較之第十三艦隊初設時更少。這可見「楊艦隊」有偏於「私兵」的傾向，與帝國軍接近，有著以司令官個人魅力為向心力的情況。

# 黑色槍騎兵：

帝國軍畢典菲爾特提督的個人艦隊，全以高速戰艦所組成，由於艦隻都塗上黑色，故稱之為「黑色槍騎兵」。在回廊會戰的前哨戰中，黑色槍騎兵與法倫海特艦隊被楊艦隊所重創，法倫海特戰死，其艦隊被編配入黑色槍騎兵。由於原屬法倫海特的艦隊一直認為是畢典菲爾特的莽進而令法倫海特戰死，故原屬法倫海特的艦隊一直與原黑色槍騎兵不太和穆。

# 雙頭鷲軍旗：

「高登巴姆王朝」的軍旗，是黑底配上金黃色的雙頭鷲。旗號在羅嚴克拉姆王朝建國後被廢棄。

# 黃金獅子旗：

「羅嚴克拉姆王朝」的軍旗，是一面以金黃色滾邊和鮮紅色底色，中央配上一頭金色的獅子像為標誌的軍旗。而亦因此旗，皇帝萊因哈特又有「金色有翼獅子」的稱號，配合這張「黃金獅子旗」，在當時和後世的人中，留下了極為強烈的印象。

# 柊館：

「柊館」是皇帝萊因哈特遷都往費沙後的臨時皇宮。名稱的由來是種植於門前兩側的柊樹，以及玄關門上刻著的柊樹徽章。宮內省曾提議把徽章改成「黃金獅子」，但最後被皇帝萊因哈特所否決。新帝國曆３年５月１４日，格里華德大公妃與懷孕的皇妃希爾德在柊館遭到地球教的恐怖行動襲擊，柊館發生大火，碰巧此時皇妃希爾德要產子，最後在帝都防衛司令官克斯拉的拯救下逃出。而皇太子亞歷山大．齊格飛．馮．羅嚴克拉姆則安全在費沙的醫院順利出生。柊館在大火中完全付之一炬，臨時皇宮遷往「貝爾塞底皇宮」。

# 貝爾塞底皇宮：

新帝國曆３年，臨時皇宮柊館在大火中完全付之一炬，臨時皇宮遷到了「貝爾塞底皇宮」。這本來是高登巴姆王朝做為高等事務官的官邸，單只以地名為名，被稱作「貝爾塞底」。皇帝萊因哈特最後便是病逝於此。

# 獅子之泉：

帝國曆４９０年、宇宙曆７９９年，萊因哈特．馮．羅嚴克拉姆公爵篡奪了「高登巴姆王朝」，創立的新王朝是為「羅嚴克拉姆王朝」，同年改行新帝國曆，是為新帝國曆１年。新帝萊因哈特計劃遷都往費沙，故下令新設的工部省工部尚書席爾瓦貝爾西，重新規劃行星費沙的都市計畫，並興建新皇宮，這就是「獅子之泉」。

遺憾的是，工部尚書席爾瓦貝爾西，於新帝國曆１年在地球教徒的恐怖行動中身亡，興建計劃因此而被延誤。另一遺憾是，在新帝國曆３年，皇帝萊因哈特還未等及完工便病逝，而這時獅子之泉還未完工，皇宮最後在新帝亞歷山大．齊格飛．馮．羅嚴克拉姆登基後，皇母希爾德的攝政時代才告完工。

# 獅子之泉七元帥：

新帝國曆３年，皇帝萊因哈特病逝前躍升繆拉、畢典菲爾特、瓦列、克斯拉、梅克林格及艾齊納哈等六位建國有功，而未陣亡的一級上將為元帥，以助其子亞力克大公日後國政。此六位元帥，聯同功績超群的開國元勳米達麥亞元帥，輔助新帝亞歷山大．齊格飛．馮．羅嚴克拉姆國政，後世通稱七位元帥為「獅子之泉七元帥」。

# 皇帝病：

正式名稱是「變異性劇症膠原病」，是一種的罕見的膠原病症，病徵是持續性原因不明的發熱，身體因發燒而不斷地消耗，致病原因相信是因為工作過勞所引起。當時來說是新病例，不單名稱是暫定，而且也未知治療方法，可說是不治之症。新皇帝萊因哈特．馮．羅嚴克拉姆病逝因為死於此症，而他也是醫學史上第一個死於此症的人，所以也被通稱為「皇帝病」。

# 影之城：

新帝國曆２年，皇帝萊因哈特遷都往費沙，為了加強費沙回廊出口的防衛，故計劃在回廊出口興建軍事要塞，這正是「影之城」和「三元帥之城」。影之城正確名稱是「夏坦布魯格」，有著「影之城」的意思，設置於費沙回廊同盟領的出口。能夠扼住回廊出入口以防衛新帝都──費沙，而且還可以作為出擊、補給、通訊的據點。第二次蘭提馬利歐會戰時，萊因哈特主陣便是集結於此。

# 三元帥之城：

新帝國曆２年，皇帝萊因哈特遷都往費沙，為了加強費沙回廊出口的防衛，故計劃在回廊出口興建軍事要塞，這正是「影之城」和「三元帥之城」。三元帥之城正確名稱是「多萊．古羅斯阿多米拉爾斯布魯格」，意味著「三元帥之城」，設置於費沙回廊帝國本土的出口，是用來紀念羅嚴克拉姆王朝死去的三名元帥──吉爾菲艾斯、法倫海特和舒坦梅茲。能夠扼住回廊出入口以防衛帝都費沙，可以作為出擊、補給、通訊的據點。

# 拉古朗市屠殺事件（染血之夜）：

公元２６９０年，處於殖民時期的銀河系，人類社會仍以地球為核心。但隨著地球逐漸衰退，各殖民星逐漸強大，地球為保護宗主權而開始鎮壓各殖民星。這時反抗地球的西留斯政府崛起，於是地球軍便向西留斯星域發動攻擊。５月１４日，地球軍攻入西留斯第六行星隆多利那的拉古朗市，對該市進行了洗劫。在當天十小時的洗劫中，拉古朗市共有九十萬市民慘遭殺害，造成損失達一百五十億個共通單位，於是５月１４日在歷史上被稱為「染血之夜」。然而，幾天後，地球軍發了宣言，認為只有二萬人死於流血事件，而且是游擊隊所為，要嫁禍地球軍，因此有必要進行大掃蕩。於是，地球軍又進行了第二次洗劫。二次洗劫使拉古朗市成為廢墟，共造成一百二十五萬人的死亡，二百五十萬人受傷，四十萬人被俘。這一事件暴露出了地球軍的凶殘臉目，從而使地球的聲望降到了極點。

# 克洛普修特克事件：

帝國曆４８６年３月２１日，在布朗胥百克公爵府舉行的宴會上，提前退席的克洛普修特克侯爵遺留下來的一只盒子發生爆炸，當場炸死十餘人，炸傷一百餘人。３月３０日，布朗胥百克公爵在得到皇帝的允許後親自討伐克洛普修特克侯爵，在經過一場苦戰後將克洛普修特克鎮壓，於５月２日返回奧丁。

在這一事件中有一小插曲。出任討伐軍戰鬥技術顧問的米達麥亞少將，擊斃了對克洛普修特克領地進行洗劫的布朗胥百克公爵的遠親而被囚禁。米達麥亞的好友羅嚴塔爾向萊因哈特求助。萊因哈特出面救出米達麥亞後，米達麥亞和羅嚴塔爾從此效忠萊因哈特。

# 格蘭多．卡納爾事件：

宇宙曆７９５年，帝國曆４８６年，第三次迪亞馬特星域會戰後，同盟軍征用了一百艘民用船運送物質，並派遣十艦巡航艦護航。但半途中調走了護航的巡航艦，只剩格蘭多．卡納爾巡航艦仍護送著民船。後來他們遭遇到帝國軍兩艘巡航艦，格蘭多．卡納爾勇敢的衝上去與敵交戰，以自己的犧牲使民船逃離，僅一艘被破壞、一艘被俘。

格蘭多．卡納爾所解救的，不僅是老百姓的生命，還包括了同盟軍的名譽。自艦長菲岡少校以下，所有戰死者都頒贈了「自由戰士勛章」。

# 卡斯特羅普動亂：

集富裕財力與權勢於一身的卡斯特羅普公爵歐以肯意外死亡，帝國財政部欲沒收其生前搜刮的不義之財，但是遭到公爵之子馬克西米利安的反抗，趕走財務省的調查官，並監禁前來勸說的瑪林道夫伯爵弗朗茲，掀起動亂。帝國軍兩次派軍鎮壓均遭失敗，修姆德提督戰死。馬克西米利安乘勢進入瑪林道夫領地。最後，應萊因哈特的要求，帝國軍派齊格飛．吉爾菲艾斯前去鎮壓。

吉爾菲艾斯派兵佯攻卡斯特羅普領地，馬克西米利安被迫解瑪林道夫之圍前去解救，在路上中吉爾菲艾斯的埋伏大敗，馬克西米利安逃出後被部下所殺，動亂平定。

在鎮壓卡斯特羅普動亂的戰役中，吉爾菲艾斯顯示出其非凡的才干，另門閥貴族為之驚愕。戰後吉爾菲艾斯晉升為中將。

# 多魯頓事件：

宇宙曆７９７年初的同盟、帝國交換俘虜結束後，楊等一行六人搭乘運載歸國兵的運輸船團前往海尼森。３月１０日在一次航行電腦的資料檢查中發現運輸船團正撞向馬斯達克里恆星。１３日，運輸船團司令官薩克斯少將經調查了解到這是航行員伊波琳．多魯頓所為，動機是在歸團兵中有一名曾欺騙過她感情的男子。當薩克斯離開向楊匯報時，多魯頓持兵器進入了緊急控制室。楊派菲列特莉加前去勸說，但無效果。１４日，楊讓一艘無人駕駛穿梭機飛離運輸船團，並告之多魯頓欺騙她感情的男子已經逃跑，多魯頓放棄讓運輸船團撞向恆星的計划後自殺，運輸船團的航行得以正常化。

由於多魯頓事件使楊晚了１０多天到達海尼森，楊告知比克古政變的可能後，比克古沒有足夠的時間制止政變。

# 海尼森廣場屠殺事件：

自由行星同盟救國軍事會議控制了海尼森後，曾發布禁止公開集會的命令。但６月２０日這天，海尼森有二十萬市民在潔西卡．愛德華的領導下於首都國家廣場舉行集會。救國軍事會議派克魯斯齊上校帶三千武裝士兵趕到廣場驅散群眾，但克魯斯齊到達後一連將包括潔西卡．愛德倫在內的數人活活打死，引起了公憤。民眾一擁而上，並奪下了克魯斯齊及其士兵的武器。於是救國軍事會議增兵鎮壓，廣場上產生暴動。

廣場屠殺事件中，有二萬市民喪生，士兵則死了一千五百名。這一事件使救國軍事會議的聲望降到了極點，自軍事和經濟上遭受重大挫折後，在政治上也產生危機，預示政變的必然失敗。

# 威斯塔朗特事件：

利普休塔特會戰開始後，布朗胥百克的侄子賽德男爵為支持伯父的霸業，變本加厲地壓榨自己的領地威斯塔朗特的民眾，引起了民眾的反抗。賽德在鎮壓失敗後，身負重傷，逃至禿鷹之城要塞後死亡。布朗胥百克大怒，不顧屬下的反對，下令對威斯塔朗特核子攻擊，告成威斯塔朗特的２００萬居民全部死亡。在攻擊前，萊因哈特曾得到消息，但出於政治上的需求，聽從奧貝斯坦的話沒有加以阻止，並拍下全過程在全國境內播放。

威斯塔朗特事件使貴族徹底失去了民心，使門閥貴族的戰敗之日提前了許多。但是，萊因哈特的置之不理使他與吉爾菲艾斯產生矛盾，並在一定程度上導致了吉爾菲艾斯的犧牲。

# 邱梅爾事件：

在萊因哈特登基後，由於首席秘書官希爾德的邀請，前往看望希爾特的久病在床的堂弟海因里希．馮．邱梅爾男爵。但邱梅爾將萊因哈特一行人誘至位於充滿杰服粒子的地下室上面的中庭進行要挾。後來因他想看萊因哈特的銀墜遭拒絕而強行去搶，被萊因哈特痛毆，爆炸控制器脫手，被搶上的奇斯里制服。在希爾德的要求下，奇斯里放開邱梅爾，邱梅爾在訴說他只是想在臨死之前做一點事的心愿後死去。此時出現了一名地球教徒欲槍殺萊因哈特，被萊因哈特的次席副官流肯擊斃。

與此同時，帝都防衛官克斯拉接到特留尼西特的密報得知地球教指使邱梅爾殺害萊因哈特，派兵掃蕩了地球教的奧丁分部，又派兵包圍了邱梅爾府，直至萊因哈特安全以後。

事件發生後，萊因哈特下達了掃蕩地球教的命令。

# 古恩．基姆．荷亞廣場事件：

宇宙曆８００年，新帝國曆２年９月１日，舊同盟政府以軍隊的相關人員和戰備兵在海尼森的古恩．基姆．荷亞廣場舉行集會，共有超過２０萬的民眾參加。帝國新領土總督羅嚴塔爾元帥沒有禁止，並派２萬帝國軍士兵維持秩序。在集會過程中，民眾反帝國意識高漲，高唱同盟國歌，高喊反帝國口號，並於１４時０６分開始向帝國軍投制石塊，帝國軍在使用瓦斯和警棍驅散民從無效後於１４時２４分向民眾開槍，結果暴動擴大，直至１５時１９分，暴動基本被控制助。這一事件中，民眾有４８４０人喪生，５００００餘人受傷，帝國軍有１１８人死亡。隨後，在舊同盟領土上又產生了不少零星的暴動。

這一事件反映出舊同盟領土的民眾對帝國統治的接受只是表面的，在民眾的內部還蔓延著很深的反帝國情緒。

# 烏魯瓦希事件：

新帝國曆２年９月２２日，萊因哈特應羅嚴塔爾邀請從費沙出發前往海尼森。原費沙自治領主魯賓斯基和內務省次長朗古懸賞策動烏魯瓦希的士兵刺殺皇帝。

１０月７日，萊因哈特到達干達爾星系的烏魯瓦希行星。２３時３０分，與萊因哈特隨行的繆拉和魯茲兩員一級上將察覺到了危險，帶萊因哈特出逃。在路上遭到了士兵們的追殺，繆拉受輕傷。同時，軍用港也遭到叛亂部隊的壓制，伯倫希爾戰艦轉移至人工湖等待萊因哈特。萊因哈特到達人工湖後，魯茲主動留下阻擊追兵。由於魯茲的出色表現，使追兵一時不敢靠前，萊因哈特等人趁機駕伯倫希爾逃離烏魯瓦希，但魯茲最後被叛亂士兵槍殺。

當羅嚴塔爾得知此事後，因不愿受奧貝斯坦與朗古的審訊，決定叛變萊因哈特，從而引發第二次蘭提馬利歐星域會戰。

# 丹亭街騷亂事件：

奧貝斯坦鎮壓海尼森動亂之後，將海尼森的共和主義分子及前同盟的重要人物全部逮捕，史稱「奧貝斯坦大掃除」，企圖以此要脅伊謝爾倫政府獻城投降，此舉遭到畢典菲爾特的反對。在爭論中，畢典菲爾特對軍務尚書大打出手，被奧貝斯坦「拘禁」，使得「黑色槍騎兵」艦隊對奧貝斯坦大有怨言。４月６日，「黑色槍騎兵」艦隊官兵與軍務尚書直局的憲兵隊產生糾紛，隨後發生斗毆，並在丹亭街對峙。後來瓦列趕到，使局勢得以控制。

# 拉格普爾監獄流血事件：

在丹亭街騷亂事件後的第１０天深夜，「奧貝斯坦大掃除」中被捕的政治犯在拉格普爾監獄發生暴動，政治犯中死亡１０８４人，受傷３０１９人。同時在海尼森傳出了「黑色槍騎兵暴動」的傳聞，使前往鎮壓監獄暴動的「黑色槍騎兵」陸戰對被憲兵隊阻攔，雙方產生沖突。另外，海尼森的市街中也出現與拉格普爾監獄暴動呼應的爆炸和放火事件，連前往鎮壓的瓦列一級上將也遭到襲擊。一連串的事件使海尼森陷入一片混亂，直至１７日７時混亂才得以控制。

後來據調查，這一事件是地球教挑起的，目的在於破壞帝國與伊謝爾倫之間成立的信賴關系。

# 柊館大火事件：

爲了阻止皇子的誕生，地球教決定刺殺皇妃希爾德。５月１４日，費沙連續１４處發生爆炸事件，使費沙的憲兵隊和帝國防禦部隊的兵力分散到各處，地球教徒則襲擊了希爾德所在的柊館。１５時，正在視察行星防衛措施的克斯拉得到報告，看穿了地球教的詭計，立即趕到柊館。此時柊館已經燃起了大火，克斯拉通過梯子進入希爾德的房間守護皇妃，並成功將希爾德救出柊館。事後，克拉斯襲擊了地球教在費沙的總部，使地球教在費沙的勢力全部覆沒。
